# Flowbit AI Dashboard

A full-stack analytics dashboard with AI-powered “Chat with Data” built using Next.js, Node.js, PostgreSQL, and Vanna AI.

---

## ⚙️ Setup

### Prerequisites
- Node.js ≥ 18
- Docker (for database)

### Run Locally

```bash
# Start backend
cd apps/api
npm install
npm run dev

# Start frontend
cd ../web
npm install
npm run dev
